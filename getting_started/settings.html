Settings

Most defaults should work without changing, however you will likely want to customize settings if you want to customize your cache location or similar.

Updating Settings

To change defaults you can either edit the settings.yml file in the installation directory at citelang/settings.yml or create a user-specific configuration by doing:

$ citelang config userinit
Created user settings file /home/vanessa/.citelang/settings.yml

You can then change a setting, such as the cache directory:

$ citelang config set cache_dir:/tmp/cache

Settings Table

The following variables can be configured in your user settings:

Title

Name

Description

Default
